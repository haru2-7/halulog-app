# ハルログ

ハルログは、1時間ごとのメモを軽く残すための個人用ログアプリです。

HTML/CSS/JavaScriptだけで動く静的Webアプリとして作っているため、GitHub Pagesなどの静的ホスティングに置いて使えます。

## 特徴

- 今日の日付を自動表示します。
- 起床、ボディバッテリー、今日の予定を記録できます。
- 7:00から22:00まで、1時間ごとのメモ欄があります。
- 入力内容は日付ごとにブラウザの `localStorage` へ自動保存されます。
- 「ログをコピー」ボタンで、記録内容をプレーンテキストとしてコピーできます。
- 入力データは外部サーバーへ送信しません。

## ファイル構成

- `index.html`: アプリ本体です。GitHub Pagesではこのファイルがトップページになります。
- `manifest.json`: ホーム画面追加時の表示設定です。
- `icons/`: 仮アイコンです。必要に応じて差し替えられます。

## GitHub Pagesで公開する場合

1. このリポジトリをGitHubに置きます。
2. リポジトリの `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. `Branch` を `main`、フォルダを `/root` にして保存します。
6. 表示されたURLをブラウザで開きます。

## iPhoneで使う場合

1. iPhoneのSafariで公開URLを開きます。
2. 共有ボタンを押します。
3. `ホーム画面に追加` を選びます。

ホーム画面から開くと、通常のWebページよりアプリに近い感覚で使えます。

## ローカルで開く場合

`index.html` をブラウザで開くと、そのまま使えます。

## データ保存について

保存キーは `halulog:YYYY-MM-DD` 形式です。日付ごとに別データとして保存されます。

データは、開いているブラウザの `localStorage` に保存されます。別の端末や別のブラウザには同期されません。
